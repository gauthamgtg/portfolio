Overview

Apex is a comprehensive, production-ready SaaS Revenue Analytics platform designed to empower startups and established companies with deep financial insights. Built with Next.js 14 and TypeScript, it centralizes subscription management, payment tracking, and investor-ready reporting into a single, intuitive dashboard.

Problem & Solution

Founders often struggle to calculate critical SaaS metrics like MRR, churn, and LTV from disparate payment records, leading to inaccurate forecasting. Apex provides a centralized solution that automatically syncs with Stripe webhooks to track subscription lifecycles, generates real-time financial dashboards, and provides downloadable investor-ready reports.

Key Features

Advanced Revenue Dashboard

Real-time visualization of metrics using Recharts for MRR, churn, and transaction history.

Subscription Management

Integrated Stripe billing support for plan upgrades, cancellations, and renewals.

Automated Reporting

Generate and download PDF invoices and financial reports for seamless accounting.

Dynamic Data Ingestion

Custom CSV upload system with column mapping for legacy data imports.

Technical Implementation

Built with **Next.js 14** and **Prisma ORM**, Apex ensures high performance and data integrity. We utilized **Neon PostgreSQL** for scalable data persistence and **Tailwind CSS** for a dense, professional UI that prioritizes information hierarchy.
