Overview

Cosmos is a sophisticated, multi-tradition astrology platform that bridges the gap between ancient celestial wisdom and modern artificial intelligence. Unlike standard astrology apps, Cosmos offers deep-dive calculations and interpretations for Western, Tamil, and Telugu systems, providing a localized and authentic experience for users across different cultures.

Multi-Tradition Architecture

The platform treats each tradition as a first-class citizen. Through the /en/, /ta/, and /te/ route prefixes, Cosmos implements independent calculation engines for Tropical and Sidereal zodiacs, ensuring that Nakshatra labels and zodiac interpretations are culturally accurate and technically precise.

Technical Implementation

Built with a high-performance Vite + React frontend and TypeScript for type-safe celestial calculations. The backend leverages Supabase (Lovable Cloud) for real-time data persistence and user authentication, while shadcn/ui provides a mystical, cosmic-themed interface with deep purples and starry gradients.
