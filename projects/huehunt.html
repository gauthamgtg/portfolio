Concept & Gameplay

HueHunt was born out of a fascination with color theory. It's a mobile-first color matching game where players must mix primary colors to match a target hue. Unlike simple games, HueHunt includes complex modes like **Hardcore (targets are hidden wÃ¤hrend mixing)** and **Memory (recalling palette sequences)**, making it a true test of visual acuity.

Smart Features

The game leverages AI to provide real-time critiques of color mixes, explaining why a specific shade is off (too much blue, lacks saturation, etc.). The **Scanner/Image Analyzer** tool allows users to pick a color from their real-world environment (via photo upload) and immediately jump into a challenge to match that exact color.

Progression & Gamification

XP & Leveling

Threshold-based progression system with unlockable badges and titles.

Streak Tracking

Daily completion goals and streak multipliers to encourage retention.

Daily Puzzles

Every player around the world gets the same seeded color puzzle per date.

Local Storage

High scores, total stats, and profile progress persisted locally without weight.

Tech Implementation

Built with **React** and the latest **Tailwind CSS (v4)** for ultra-fast styling. The performance is optimized for mobile browser runtimes. Using **Capacitor**, the PWA is wired for native Android packaging, allowing it to sit on the home screen with a standalone feel.
